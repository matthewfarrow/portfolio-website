## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --stone-dark: #2a2a2a;
    --gold-accent: #d4af37;
    --blue-glass: #4a90e2;
    /* etc. */
}
```

### Content
Update the content directly in `index.html` to add your own:
- Projects and portfolio items
- Social media links
- Articles and writings
- Personal information
